# Dokumentasi
- Tampilkan data
  GET url/{Controller}
**- Tampilkan data by id**
  GET url/{Controller}/{id}
**- Tambah data**
  POST url/{Controller}
**- Hapus data**
  DELETE url/{Controller}
**- Ubah data**
  PUT url/{Controller}
